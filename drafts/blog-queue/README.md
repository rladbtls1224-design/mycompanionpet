# Blog Publish Queue

2026-07-20 공개 사이트와 로컬 콘텐츠를 함께 확인해 기존 게시 글과 겹치지 않도록 구성한 14개 추천 큐입니다.

사용자가 `1번 배포해줘`, `3번, 4번 배포해줘`처럼 요청하면 아래 번호를 `queue.json`의 동일 번호와 연결합니다. 해당 글의 Astro Markdown과 이미지 3~4개를 완성하고 검수한 뒤 빌드, 커밋, push 순서로 배포합니다.

## 배포 날짜 규칙

- 새 글을 배포할 때 `pubDate`와 `updatedDate`는 배포 실행일의 한국 시간 날짜로 맞춥니다.
- 기존 글을 수정 배포할 때는 원래 `pubDate`를 유지하고 `updatedDate`만 수정 배포일로 갱신합니다.
- `npm run publish:queued -- {번호}`도 새 글의 두 날짜를 자동으로 배포 당일에 맞춥니다.
- 이미지 생성, 의료 표현, 내부 링크, 중복 검사와 `npm run build`가 끝나기 전에는 push하지 않습니다.

## 추천 순서

1. 강아지가 포도 한 알 먹었을 때 어떻게 해야 할까? 위험 신호와 대처 순서
2. 고양이 양파 먹으면 안 되는 이유｜익힌 양파·국물·가루까지 주의
3. 강아지 닭가슴살 먹어도 될까? 삶는 법과 주식 대체 주의점
4. 고양이 계란 먹어도 될까? 완숙 흰자·노른자 급여 기준
5. 강아지 당근 먹어도 될까? 생당근과 익힌 당근, 조각 크기 비교
6. 고양이 연어 먹어도 될까? 익힌 연어와 훈제연어·가시 주의
7. 강아지 바나나 먹어도 될까? 당분·껍질·급여량 기준
8. 고양이 사과 먹어도 될까? 씨와 심, 껍질 제거 기준
9. 강아지 단호박 먹어도 될까? 찐 단호박과 껍질·씨 주의
10. 고양이 오이 먹어도 될까? 껍질·씨·수분 간식 기준
11. 강아지 브로콜리 먹어도 될까? 송이와 줄기, 익혀 주는 법
12. 고양이 소고기 먹어도 될까? 익힘 정도와 기름·양념 주의
13. 강아지 연어 먹어도 될까? 익힌 연어와 생연어·가시 주의
14. 고양이 블루베리 먹어도 될까? 냉동 블루베리와 급여량

## 큐 파일

SEO/GEO 메타 정보, slug, 썸네일 콘셉트, 내부 링크 후보와 우선순위 근거는 `drafts/blog-queue/queue.json`에 저장되어 있습니다. `status: "planned"`인 항목은 배포 요청을 받았을 때 본문과 이미지를 생성한 뒤 게시합니다.
