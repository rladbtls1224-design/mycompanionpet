# Blog Publish Queue

2026-07-12 기준으로 기존 게시 글과 겹치지 않게 다시 구성한 14개 추천 큐입니다.

사용자가 `1번 배포해줘`, `3번, 4번 배포해줘`처럼 요청하면 이 큐의 번호 순서대로 글을 작성하고 이미지를 생성한 뒤 게시합니다. 실제 배포 시에는 글의 `pubDate`를 배포 당일 날짜로 맞춥니다.

## 배포 날짜 규칙

- 초안을 게시 파일로 복사할 때 `pubDate`는 실행일의 한국 시간 날짜로 자동 갱신합니다.
- 직접 배포 작업을 할 때도 같은 규칙을 적용합니다.
- 예: 2026-07-12에 배포하면 `pubDate: "2026-07-12"`를 사용합니다.

## 추천 순서

1. 강아지 오이 먹어도 될까? 수분 간식으로 줄 때 양과 주의점
2. 고양이 참치캔 먹어도 될까? 사람용 참치와 고양이용 간식 차이
3. 강아지 치즈 먹어도 될까? 락토스와 나트륨을 먼저 확인하세요
4. 고양이 닭가슴살 먹어도 될까? 삶은 닭가슴살 급여 기준
5. 강아지 사과 먹어도 될까? 씨와 심 제거가 먼저입니다
6. 고양이 요거트 먹어도 될까? 무가당이어도 조심해야 하는 이유
7. 강아지 양배추 먹어도 될까? 생양배추와 익힌 양배추 차이
8. 고양이 사료 토핑 해도 될까? 입맛보다 영양 균형이 먼저입니다
9. 강아지 사료 토핑 괜찮을까? 편식 만들지 않는 기준
10. 고양이 변비에 식단이 중요할까? 수분과 식이섬유를 보는 법
11. 강아지 변이 묽을 때 사료를 바꿔야 할까? 식단 점검 순서
12. 고양이 사료 교체는 며칠에 걸쳐야 할까? 7일 전환 기준
13. 강아지 사료 교체 방법, 갑자기 바꾸면 안 되는 이유
14. 노령묘에게 습식사료가 더 좋을까? 수분과 씹기 편의 기준

## 큐 파일

자세한 SEO/GEO 메타 정보, slug, 썸네일 콘셉트, 내부 링크 후보는 `drafts/blog-queue/queue.json`에 저장되어 있습니다.
