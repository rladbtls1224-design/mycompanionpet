# Quality Checklist

새 글 작성 후 확인한다.

- [ ] 대상 동물이 명확한가?
- [ ] 강아지 글에 고양이 내용이 섞이지 않았는가?
- [ ] 고양이 글에 강아지 내용이 섞이지 않았는가?
- [ ] 도입부가 기존 글과 다르게 시작하는가?
- [ ] 검색자의 핵심 질문에 초반에 답했는가?
- [ ] 한눈에 보기 표가 있는가?
- [ ] 주제에 맞는 표가 있는가?
- [ ] 급여량이 절대 기준처럼 표현되지 않았는가?
- [ ] 위험 증상과 수의사 상담 기준이 있는가?
- [ ] 실제 상황 예시가 3개 이상인가?
- [ ] FAQ가 5개인가?
- [ ] 참고자료 문단이 있는가?
- [ ] 이미지 alt가 구체적인가?
- [ ] 내부 링크 후보가 같은 동물 카테고리인가?
- [ ] 글만의 새로운 정보가 3개 이상인가?
