# SEO Checklist

- [ ] SEO 제목이 1개만 최종 선택되었는가?
- [ ] 제목이 검색 의도를 반영하는가?
- [ ] 메타 설명이 80~150자 수준인가?
- [ ] slug가 영문 소문자와 하이픈으로 되어 있는가?
- [ ] H1은 1개인가?
- [ ] H2 구조가 자연스러운가?
- [ ] 키워드를 억지로 반복하지 않았는가?
- [ ] 본문 초반에 직접 답변이 있는가?
- [ ] FAQ가 실제 검색 질문처럼 작성되었는가?
- [ ] 내부 링크가 토픽 클러스터에 맞는가?
- [ ] 이미지 파일명이 SEO 친화적인가?
- [ ] alt 텍스트가 구체적인가?
- [ ] 중복 위험이 낮은가?
