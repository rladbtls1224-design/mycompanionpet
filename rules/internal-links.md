# 내부 링크 규칙

## 기본 원칙

내부 링크는 사이트 전체의 토픽 클러스터를 강화하기 위해 사용한다.

## 강아지 글

강아지 글에서는 강아지 관련 글만 추천한다.

우선순위:

1. 같은 음식군 글
2. 같은 위험도 글
3. 같은 급여 상황 글
4. 강아지 영양 허브
5. 강아지 간식/급여법 허브

예:

강아지 참외 글의 내부 링크 후보:

- 강아지 수박 먹어도 될까?
- 강아지 멜론 먹어도 될까?
- 강아지 복숭아 먹어도 될까?
- 강아지 바나나 먹어도 될까?
- 강아지 포도 먹으면 안 되는 이유

## 고양이 글

고양이 글에서는 고양이 관련 글만 추천한다.

예:

고양이 복숭아 글의 내부 링크 후보:

- 고양이 수박 먹어도 될까?
- 고양이 블루베리 먹어도 될까?
- 고양이 바나나 먹어도 될까?
- 고양이 사과 먹어도 될까?
- 고양이 포도 먹으면 안 되는 이유

## 실제 링크 삽입 방식

기존 파일이 있는 경우 실제 URL로 연결한다.
기존 파일이 없는 경우 `내부 링크 후보`로만 표시하고, 임의 URL을 만들지 않는다.

예:

```md
## 함께 읽으면 좋은 글

- [강아지 수박 먹어도 될까?](/blog/can-dogs-eat-watermelon/)
- 강아지 멜론 먹어도 될까? — 후속 작성 추천
```

## 링크 개수

본문 내부에는 2~4개 자연 삽입.
글 하단에는 5개 추천.

## 중복 방지

모든 글 하단에 매번 같은 내부 링크 5개를 넣지 않는다.
주제와 가장 가까운 글을 우선한다.
