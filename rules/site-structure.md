# 사이트 구조 규칙

이 사이트는 강아지 영양과 고양이 영양을 명확히 분리한다.

## 최상위 카테고리

- 강아지 영양
- 고양이 영양

## 강아지 영양 하위 클러스터

- 강아지 과일
- 강아지 채소
- 강아지 육류
- 강아지 생선
- 강아지 간식
- 강아지 독성 음식
- 강아지 영양제
- 강아지 급여 방법
- 강아지 연령별 영양
- 강아지 질환별 식이 주의

## 고양이 영양 하위 클러스터

- 고양이 과일
- 고양이 채소
- 고양이 육류
- 고양이 생선
- 고양이 간식
- 고양이 독성 음식
- 고양이 영양제
- 고양이 급여 방법
- 고양이 연령별 영양
- 고양이 질환별 식이 주의

## 내부 링크 원칙

강아지 글은 강아지 글끼리 연결한다.
고양이 글은 고양이 글끼리 연결한다.

예외적으로 사이트 전체 안내 페이지나 개인정보처리방침 같은 공통 페이지는 연결 가능하다.

## 허브 페이지 연결

강아지 글이면 아래 허브를 우선 추천한다.

- 강아지 영양
- 강아지 과일
- 강아지 음식
- 강아지 간식

고양이 글이면 아래 허브를 우선 추천한다.

- 고양이 영양
- 고양이 과일
- 고양이 음식
- 고양이 간식

## 새 글 작성 시 클러스터 판단

Codex는 주제를 받으면 먼저 아래를 판단한다.

1. 대상 동물: 강아지 또는 고양이
2. 음식 유형: 과일, 채소, 육류, 생선, 간식, 독성 음식, 영양제 등
3. 위험도: 안전, 소량 가능, 주의 필요, 금지
4. 연결할 허브
5. 연결할 기존 글
6. 다음에 작성할 후속 글
