# 콘텐츠 중복 방지 규칙

이 사이트는 앞으로 수백 개 이상의 글을 발행한다. 따라서 음식 이름만 바꾼 반복 콘텐츠는 절대 만들지 않는다.

## 새 글 작성 전 반드시 할 일

1. 기존 글 파일 목록을 확인한다.
2. 같은 동물/같은 음식군 글을 확인한다.
3. 기존 도입부, FAQ, 표, 예시, 체크리스트와 겹칠 수 있는 부분을 피한다.
4. 새 글만의 차별점 5개를 먼저 정한다.
5. 본문에 새 글만의 정보 3개 이상을 포함한다.

## 중복 금지 항목

- 같은 도입 문장
- 같은 문단 구조
- 같은 FAQ 질문
- 같은 FAQ 답변
- 같은 체크리스트
- 같은 실제 상황 예시
- 같은 이미지 배경
- 같은 이미지 프롬프트
- 같은 표 형식만 반복
- 같은 결론 문장
- 같은 주의사항 문장 복붙

## 차별화 방법

같은 과일 글이라도 관점을 다르게 잡는다.

예:

- 참외: 여름철 수분, 씨와 껍질, 당분
- 수박: 수분 보충, 씨 제거, 냉장 급여
- 복숭아: 씨, 통조림, 껍질, 향에 대한 관심
- 바나나: 탄수화물, 당분, 변 상태
- 블루베리: 항산화 성분, 크기, 과식
- 사과: 씨, 껍질, 조각 크기

## 도입부 차별화 예시

- 계절 상황: 여름, 겨울, 명절
- 장소 상황: 주방, 캠핑, 산책 후, 카페
- 행동 상황: 바닥에 떨어진 것을 먹음, 아이가 줌, 냄새 맡고 다가옴
- 건강 상황: 노령, 비만, 소화기가 예민함

## 표 차별화 예시

- 어떤 글은 영양 성분 표
- 어떤 글은 급여 가능 부위 표
- 어떤 글은 비슷한 음식 비교표
- 어떤 글은 위험도 표
- 어떤 글은 상황별 판단표

## FAQ 차별화 규칙

FAQ는 실제 검색 의도를 반영하되, 매번 다른 질문을 만든다.

반복하기 쉬운 질문:

- 먹어도 되나요?
- 얼마나 먹어도 되나요?

이 질문만 반복하지 말고 다음을 섞는다.

- 껍질은 괜찮나요?
- 씨를 삼켰다면 어떻게 하나요?
- 냉동으로 줘도 되나요?
- 통조림은 괜찮나요?
- 말린 것은 괜찮나요?
- 새끼도 가능한가요?
- 노령이면 주의할 점이 있나요?
- 당뇨가 있으면 피해야 하나요?
- 알레르기 반응은 어떻게 보나요?

## 최종 점검

Duplicate Risk는 내부 검토용으로만 판단하고, 게시 본문에는 표시하지 않는다.

- LOW: 기존 글과 구조, 예시, FAQ가 충분히 다름
- MEDIUM: 일부 유사하지만 주제상 불가피함
- HIGH: 재작성 필요
