# 글쓰기 규칙

## 독자

대상 독자는 반려동물을 키우는 일반 보호자다. 수의학 지식이 없어도 이해할 수 있게 쓴다.

## 기본 문체

- 친절하지만 지나치게 가볍지 않게 쓴다.
- 문장은 짧고 명확하게 쓴다.
- 어려운 용어는 쉽게 풀어 설명한다.
- 광고성 표현을 피한다.
- AI가 쓴 것처럼 반복되는 문장을 피한다.
- 같은 접속사와 같은 문장 구조를 반복하지 않는다.

## 본문 길이

- 기본 본문은 최소 2,500자 이상을 목표로 한다.
- 단, 불필요한 반복으로 길이를 늘리지 않는다.

## 도입부 규칙

도입부는 검색자가 실제로 겪는 상황에서 시작한다.

나쁜 예:

- 반려동물을 키우는 보호자라면 한 번쯤 궁금해했을 것입니다.
- 요즘 많은 사람들이 궁금해합니다.

좋은 예:

- 여름에 참외를 깎다 보면 옆에서 기다리는 강아지에게 한 조각 줘도 되는지 고민될 수 있습니다.
- 복숭아를 먹다가 고양이가 냄새를 맡고 다가오면, 과육보다 씨와 껍질이 먼저 걱정됩니다.

## 본문 필수 구성

1. 한눈에 보기
2. 도입부
3. 핵심 개념 설명
4. 영양 또는 성분 표
5. 급여 가능 여부
6. 급여량 예시
7. 급여 시 주의할 점
8. 보호자가 자주 하는 오해
9. 실제 상황별 예시
10. 보호자 체크리스트
11. 수의사 상담이 필요한 경우
12. 결론
13. 참고 자료
14. FAQ
15. 함께 읽으면 좋은 글
SEO 자가검토는 본문 구성에 포함하지 않고 작성 후 내부 검토용으로만 사용한다.
다음에 쓰면 좋은 글은 게시 본문에 노출하지 않는다. 작성자에게만 별도 제안으로 보여준다.
## 한눈에 보기 규칙

본문 초반에 검색 의도를 빠르게 해결하는 요약 박스를 둔다.

예시:

```md
## 한눈에 보기

| 항목 | 내용 |
|---|---|
| 급여 가능 여부 | 건강한 성견 기준으로 소량 가능 |
| 주의할 부분 | 씨, 껍질, 과식 |
| 급여 방식 | 작게 자른 과육만 소량 |
| 피해야 할 경우 | 당뇨, 비만, 소화기 질환, 알레르기 의심 |
| 상담 필요 | 구토, 설사, 무기력, 씨 섭취 |
```

## 영양 성분 표 규칙

영양 성분 한눈에 보기 섹션은 표로 작성한다. 항목, 특징, 보호자가 볼 점처럼 비교하기 쉬운 열을 사용하고 긴 문장은 짧게 나눈다.

## 표 규칙

표는 매번 같은 형식만 쓰지 않는다. 주제에 맞게 선택한다.

가능한 표 유형:

- 영양 성분 표
- 급여 가능 부위 표
- 급여량 예시 표
- 위험도 비교 표
- 비슷한 음식 비교 표
- 상황별 판단 표

## 실제 상황 예시 규칙

최소 3개 이상 작성한다. 매 글마다 상황을 다르게 구성한다.

가능한 상황:

- 산책 후
- 여름철 간식
- 캠핑이나 여행 중
- 아이가 몰래 준 경우
- 식탁에서 떨어진 것을 먹은 경우
- 냉장 또는 냉동 과일
- 말린 과일
- 통조림 과일
- 노령 반려동물
- 새끼 반려동물
- 비만 또는 당뇨가 있는 경우

## 결론 규칙

결론은 짧게 정리하되, 수의사 진료를 대체하지 않는 일반 정보성 콘텐츠라는 점을 자연스럽게 포함한다.
