# 반려동물 영양 블로그 글 생성 프롬프트

아래 입력값을 바탕으로 Astro Markdown 블로그 본문을 작성한다.

## 입력값

- 제목: `{{title}}`
- 글 유형: `{{animalType}}`
- 발행일: `{{pubDate}}`
- 추천 slug: `{{slug}}`

## 작성 목표

반려동물 보호자가 검색을 통해 들어왔을 때 바로 이해하고 판단할 수 있는 정보성 글을 작성한다. 글은 친절하고 차분한 한국어 문체를 사용한다. SEO를 고려하되 키워드를 억지로 반복하지 않는다.

## 반드시 포함할 본문 구성

1. 도입부
2. 핵심 개념 설명
3. 해당 동물 기준 영양 정보
4. 먹어도 되는지 또는 급여 가능 여부
5. 급여 시 주의할 점
6. 적정 급여량 또는 판단 기준
7. 보호자가 자주 하는 오해
8. 실제 상황별 예시
9. 보호자 체크리스트
10. 수의사 상담이 필요한 경우
11. 결론
12. FAQ 5개
13. 내부 링크 추천 5개
14. 후속 글 주제 추천 5개
15. 이미지 제작 메모 3~5개

## 이미지 규칙

본문 중간중간에 Markdown 이미지 문법으로 이미지 위치를 넣는다.

예:

```markdown
![강아지 참외 급여 시 주의사항](dog-korean-melon-feeding-caution.webp)
```

이미지는 본문 2,000자 기준 3~5개 추천한다. 이미지 유형은 아래 중에서 선택한다.

- 대표 이미지
- 급여 예시 이미지
- 주의사항 인포그래픽
- 체크리스트 이미지
- 상황별 예시 이미지

본문 하단의 이미지 제작 메모에는 각 이미지별로 아래 항목을 정리한다.

- 삽입 위치
- 이미지 용도
- 이미지 설명
- alt 텍스트
- 이미지 파일명 추천
- 이미지 생성 프롬프트

## 안전 표현 규칙

- 특정 질병을 진단하거나 치료를 단정하지 않는다.
- `무조건 좋다`, `완치된다`, `반드시 먹여야 한다`, `이것만 먹이면 된다` 같은 표현을 쓰지 않는다.
- 위험 증상, 질병 가능성, 장기적인 식단 변경이 필요한 경우 수의사 상담을 권장한다.
- 일반 정보 제공용 콘텐츠이며 수의사 진료를 대체하지 않는다는 취지의 문장을 결론 근처에 자연스럽게 포함한다.

## 출력 형식

frontmatter 없이 Markdown 본문만 출력한다. H2/H3 제목, 목록, 표를 적절히 사용한다.
