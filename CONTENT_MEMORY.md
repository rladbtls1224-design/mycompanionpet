# Content Memory

이 파일은 기존 글의 특징을 기록하여 중복을 줄이기 위한 메모입니다.
새 글을 발행할 때마다 아래 형식으로 추가합니다.

## 예시

### 강아지 참외

- slug: can-dogs-eat-korean-melon
- category: 강아지 영양
- cluster: 강아지 과일
- 도입부 상황: 여름에 보호자가 참외를 깎는 상황
- 핵심 표: 영양 성분 표
- 주요 주의사항: 씨, 껍질, 당분, 과식
- 실제 예시: 산책 후, 아이가 준 경우, 노령견
- FAQ 주제: 씨, 껍질, 냉장 급여, 당뇨, 새끼 강아지
- 이미지 배경: 주방, 식탁, 체크리스트
- 다음 글에서 피할 것: 같은 여름 도입부, 같은 산책 후 예시 반복
