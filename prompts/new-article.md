# 새 글 작성용 Codex 프롬프트

아래 프롬프트를 Codex에 입력한다.

```txt
주제: {예: 강아지 참외}

작업:
1. 기존 Astro 프로젝트 구조와 기존 블로그 글을 먼저 확인해줘.
2. 이 주제가 강아지 글인지 고양이 글인지 판별해줘.
3. 기존 글과 중복되지 않도록 차별점 5개를 먼저 정리해줘.
4. Astro Markdown 글 파일을 생성해줘.
5. frontmatter, 본문, FAQ, 참고자료, 내부 링크 후보, 후속 글 후보를 포함해줘.
6. 이미지 3~4개의 삽입 위치, alt, 파일명, 영어 이미지 생성 프롬프트를 포함해줘.
7. 이미지 파일은 실제로 생성하지 말고 필요한 파일 경로와 TODO를 남겨줘.
8. 가능하면 npm run build로 빌드 오류를 확인해줘.
9. Git commit/push는 내가 따로 요청하기 전에는 하지 마.
```
